## 如何安装模组

当前模组的最新正式版版本为：`2.4.1`
本模组适用的版本如下：

<Card title="2.4.1 版本">

- Forge: `1.20.1` `1.19.2` `1.18.2` `1.16.5`
- Fabric: `1.21.1` `1.21`
  `1.20.1` `1.19.2` `1.18.2` `1.16.5`
- NeoForge: `1.21.1` `1.21`

</Card>

## 如何打开模型选择页面

在游戏中，按下 `Alt Y` 组合键（Forge 和 NeoForge 版）或者 `Y` 键（Fabric 版），若操作成功，将会显示模型界面。
若没有反应，请检查以下问题：

- 您是否已经安装了此模组？
  - 请确认您安装的模组是否与 Minecraft 和 Mod 加载器的版本对应。
  - 注意区分 Forge 版和 Fabric 版的差异！
- 您是否按下了正确的键位？
- 您是否修改了默认键位？
- 键位是否存在冲突？
- 您的键盘是否正常？

## 如何安装模型文件

<Aside>

下文提及的模型安装仅适用于单人游戏。如果您是服务器玩家，需要服主才能安装模型！

</Aside>

<Steps>

1. 打开模型选择界面后，点击页面右上角的`打开模型文件夹`按钮。

   ![1.jpg](https://s2.loli.net/2025/01/31/1ClHjr3ApLx5u6m.png)

2. 在弹出的界面中点击`打开模型文件夹`。

   ![2.jpg](https://s2.loli.net/2025/01/31/rTOg4kZLb3jpzPn.png)

3. 此时会弹出一个文件夹窗口，如下图所示：

   ![3.jpg](https://s2.loli.net/2025/01/31/T6cVnUyfmi927GW.png)

4. 将您获取到的模型文件放入弹出的窗口中，如图所示：

   ![4.jpg](https://s2.loli.net/2025/01/31/VuiRdxMIFLvpNBt.png)

   ![5.jpg](https://s2.loli.net/2025/01/31/rThBdgqjwlp9c24.png)

   完成后，返回游戏。

5. 按下键盘上的 `/` 键，输入指令 `ysm model reload`（该指令需要 OP 权限），如下图所示：

   ![6.jpg](https://s2.loli.net/2025/01/31/OUp6GX35SfdDezJ.png)

6. 按下回车键执行，当弹出以下消息时，说明模型加载完成：

   ![7.jpg](https://s2.loli.net/2025/01/31/56WfVpFUtXidmKD.png)

7. 此时打开模型选择页面，您应该能看到刚刚安装的文件。

   ![8.jpg](https://s2.loli.net/2025/01/31/FahgjmoczbBH3Rf.png)

   若模型较多，您可能需要使用下方的翻页按钮来查找您的模型。

</Steps>

## 模型无法找到

若您已按上文说明放置模型，但仍然无法找到模型，请检查以下问题：

- 请确认您是否安装了 YSM 模组？
- 请确认您安装的 YSM 版本是否为最新版（最新版为 2.4.1）。绝大多数新模型需要 2.2.2 及以上的版本才能运行，1.1.5 版本将不再支持！
- 请确认您是在单人游戏，而非多人游戏，多人游戏需要服主在服务器安装后才能使用。
- 请确认您是否使用了指令 `/ysm model reload` 来重载模型。
- 如果您是第一次游玩，并且安装的模型较多，在安装完重载后可能会只显示默认模型。此时，请耐心等待一会，模组正在为模型建立缓存，缓存建立完成后模型将会显示。此时再次打开模型选择界面即可看到模型。（多人游戏同理）

## 始终显示的是原版玩家模型

如果您已经安装了 YSM 模组，并且能够打开模型选择界面，但无论使用什么模型都只显示原版玩家模型，请在模组设置中关闭`阻止渲染自身玩家模型`，关闭方法如下：

<Steps>
1. 打开模型选择界面：

    若您不知道如何打开，请查看[如何打开模型选择页面？](#如何打开模型选择页面)

2.  点击`打开模型配置`按钮：

    ![1.jpg](https://s2.loli.net/2025/01/31/kGebx86qE7lIfzW.png)

3.  取消图示勾选：

    ![2.jpg](https://s2.loli.net/2025/01/31/riqge8GDN95b1QK.png)

</Steps>

## 模型加载后显示异常

![optifine.jpg](https://s2.loli.net/2025/01/31/rHI5iT4zF6OsxvX.jpg)

此时，您有两种解决方案：

[Iris（Fabric）](https://www.mcmod.cn/class/3697.html) 这样的光影加载器。

</Card>

使用模组自带的兼容渲染器，方法如下：

<Steps>

1. 打开模型选择界面：

2. 点击`打开模型配置`按钮：

   ![1.jpg](https://s2.loli.net/2025/01/31/kGebx86qE7lIfzW.png)

## 提示“不支持的 YSM 模型版本”

![lowversion.jpg](https://s21.ax1x.com/2025/02/03/pEZotl6.jpg)

目前，YSM 已经推出了官方的网易中国版，您可以在网易组件商城搜索`是，史蒂夫模型`，免费下载该组件进行游玩！

## 手机能否游玩该模组

可以，但仅限 Java 版 1.1.5 及以下版本。从 1.2.0 版本开始，由于内核代码的更替，暂时无法支持手机游玩。我们会在未来的版本中逐步解决此问题。
您也可以选择网易版进行游玩。

## 如何移动额外玩家渲染的位置

在游戏里，按下 `Alt P` 组合键（Fabric 版本下只需按 `P` 键），即可打开额外玩家渲染位置设置界面。

在此界面中，您可以依据右上角给出的说明，对额外玩家渲染的位置、大小以及角度进行修改。若您完成设置，按下 `Esc` 键即可退出配置界面。

![move.jpg](https://s2.loli.net/2025/01/31/2DV7IORtHzgJT93.png)
