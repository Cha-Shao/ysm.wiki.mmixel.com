Yes Steve Model 是一款用于修改原版玩家模型的模组，它采用了 Minecraft 基岩版的模型和动画文件，并且具备诸多优化以及模型加密功能。

![img](https://s2.loli.net/2023/01/01/RAor58n6LBct3kW.jpg)

<Aside type="tip">

目前，Yes Steve Model 的最新版本号为 **2.4.1**。

</Aside>

<Aside type="tip">

**2.4.1 版本** 同时支持 1.16.5/1.18.2/1.19.2/1.20.1 Forge，1.16.5/1.18.2/1.19.2/1.20.1/1.21/1.21.1 Fabric 以及 1.21/1.21.1 NeoForge，共计十二个版本。

</Aside>

## 基本介绍

Yes Steve Model 模组充分考虑了服务器群体对模组的需求，采用了众多适用于服务器环境的设计，具体如下：

- 客户端模型自动同步：普通玩家进入服务器时，服务器会**自动**将模型同步到玩家的电脑上。
- 模型文件加密：发送到玩家客户端的模型均为**加密的二进制文件**，能有效避免模型被窃取的问题。
- 模型权限功能：模型可单独设置授权，只有 OP 输入指令授权后，特定模型才能够被使用。
- 原版玩家模型修改：添加了默认的 Steve 和 Alex 模型，这两个模型都能自动调用玩家皮肤进行显示。
- 动画轮盘功能：按下 `Z` 键，可打开一个轮盘动画，通过它能方便地播放各种有趣的动画（如动作、表情等）。
- 女仆模组渲染支持：使用 2.4.0 及后续版本的 Yes Steve Model 模组和 1.2.0 版本之后的 Touhou Little Maid 模组，即可为女仆应用相应的模型和动画。

## 模组兼容说明

- 1.16.5/1.18.2 Forge 版本增加了对[永恒枪械工坊](https://www.curseforge.com/minecraft/mc-mods/timeless-and-classic-guns-tac)模组的兼容，但需要 **0.3.7** 及以上版本的[永恒枪械工坊](https://www.curseforge.com/minecraft/mc-mods/timeless-and-classic-guns-tac) 模组，否则游戏内持枪会导致游戏崩溃。
- 1.18.2/1.19.2/1.20.1 Forge 版本增加了对[永恒枪械工坊：零](https://www.curseforge.com/minecraft/mc-mods/timeless-and-classics-zero)模组的兼容。
- 尽管增加了对[更真实的第一人称模型](https://www.curseforge.com/minecraft/mc-mods/first-person-model) 模组的兼容，但仍存在一些错误。
- 增加了对 Oculus(Forge) 和 Iris(Fabric) 模组 PBR 材质的支持。
- 增加了对[拔刀剑](https://www.curseforge.com/minecraft/mc-mods/slashblade)动画的支持，2.2.1 版本还支持[拔刀剑：重锋](https://www.curseforge.com/minecraft/mc-mods/slashblade-resharped)。2.4.0 版本支持部分日式附属的刀技动画。
- 支持[时装盔甲](https://www.curseforge.com/minecraft/mc-mods/cosmetic-armor-reworked)的盔甲和鞘翅插槽。
- 支持 [SWEM（马术）](https://www.curseforge.com/minecraft/mc-mods/swem) 和 [Parcool（跑酷）](https://www.curseforge.com/minecraft/mc-mods/parcool) 模组。
- 兼容 [Carpet](https://www.curseforge.com/minecraft/mc-mods/carpet) 和 [Curtain](https://www.curseforge.com/minecraft/mc-mods/curtain) 模组的假人。
- 兼容 [Touhou Little Maid](https://www.curseforge.com/minecraft/mc-mods/touhou-little-maid)，可为女仆替换本模组的渲染模型。需要使用 1.2.0 版本以上的女仆模组。
- 兼容 [精妙背包](https://modrinth.com/mod/sophisticated-backpacks)，可渲染该模组的背包。

## 平台支持情况

1.1.5 及之前的版本采用纯 Java 编写，支持三大主流平台和手机平台。

由于旧版 Java 加密容易被破解，从 1.2.0 版本开始，核心的加密和渲染部分均采用 C++ 编写，这大大提高了渲染效率和安全性，但多平台支持情况欠佳。

目前，客户端仅支持 Windows x64 系统，服务端支持 Windows 和 Linux x64 系统。对其他平台和架构的支持也在逐步推进中，后续有完成进度时会及时公告。
