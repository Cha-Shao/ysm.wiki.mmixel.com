Yes Steve Model 需在客户端和服务端同时安装，方可正常使用全部功能。

若你**仅在客户端安装**，需先进入单人游戏，待模型加载完毕后，才可在服务器使用！

## 平台及虚拟机支持情况

1.1.5 及之前的版本，支持 Windows、Linux、MacOS、安卓、iOS（Pojav 等相关启动器）等各大主流平台。不过从 1.2.0 版本起，后续版本暂时仅支持 Windows（客户端和服务端）和 Linux（服务端），对其他平台的支持仍在开发中。

我们对 Oracle、Azul Zulu、微软、Dragonwell 等虚拟机进行了 JVM 测试。若你在运行 YSM 时出现 `Exception message: java.lang.RuntimeException: err: 54` 错误，更换上述几种 JVM 即可。

- 微软 OpenJDK<Badge type="tip" text="推荐" />：[https://learn.microsoft.com/zh-cn/java/openjdk/download](https://learn.microsoft.com/zh-cn/java/openjdk/download)
- Oracle JVM：[https://www.oracle.com/cn/java/technologies/downloads/](https://www.oracle.com/cn/java/technologies/downloads/)
- Azul Zulu：[https://www.azul.com/downloads/](https://www.azul.com/downloads/)
- 阿里云 Dragonwell：[https://www.aliyun.com/product/dragonwell](https://www.aliyun.com/product/dragonwell)

## NeoForge / Forge 版本的说明

玩家首次进入服务器后，等待十几秒（这是模型同步的耗时，具体时长会根据服务器模型数量的多少而有所不同），便可使用快捷键 `Alt` + `Y` 打开如下 GUI 界面：

![1.png](/images/Ya7DMmKToSneN6L.png)

**① 模型切换按钮**：点击该按钮，可切换为对应的模型。若按钮背景为灰色，则表明该模型未授权。左上角的数字代表该模型可切换的材质数量。

**② 模型类别切换按钮**：能够在所有模型、已授权模型、收藏模型之间进行切换。

**③ 详情界面按钮**：点击此按钮，可进入模型详情界面，查看模型的所有可用材质和动画。中间的预览窗口支持操作：使用鼠标左键拖拽可旋转模型，使用鼠标右键拖拽可移动模型，滚动鼠标滚轮可缩放模型。点击右侧的材质选择框，可选择不同的材质；点击左侧的动画列表，可预览播放不同的动画。

![3.png](/images/UxtCNy9wEg1XjSI.png)

## Fabric 版本的说明

<Aside type="tip">

Fabric 的迁移工作由**番茄布丁**（[https://github.com/TomatoPuddin](https://github.com/TomatoPuddin)）全力完成！让我们向他致以崇高的敬意！

</Aside>

Fabric 版本的绝大部分功能与 Forge 版本**完全一致**，并且已与 Sodium、Sodium Extra、Iris、Lithium、ModernFix、FerriteCore、ServerCore、Debugify、Carpet（含 FakePlayer 功能）等模组进行了兼容性测试，稳定性良好。

Fabric 版本需要两个运行前置条件：

- [Fabric Api](https://www.mcmod.cn/class/3124.html)
- [Forge Config Api Port](https://www.mcmod.cn/class/5510.html)（2.2.1 版本起不再需要安装）

同时，它与以下两个模组存在兼容互动内容：

- [First - Person Model](https://www.mcmod.cn/class/4391.html)（添加了更好的第一人称视角兼容性）
- [Amecs](https://www.mcmod.cn/class/2003.html)（支持与 Forge 版本一致的组合快捷键）
- [Modern KeyBinding](https://www.curseforge.com/minecraft/mc-mods/modern-keybinding-fabric)（另一款支持与 Forge 版本一致的组合快捷键，自 2.2.1 版本起开始支持）

Fabric 仅在按键方面与 Forge 版略有不同。由于 Fabric 的按键绑定接口不支持 Ctrl / Alt / Shift 组合键，因此需要安装 [Amecs](https://www.mcmod.cn/class/2003.html) 模组才能支持。但该模组与常用模组 [Controlling](https://www.mcmod.cn/class/1191.html) 存在冲突，所以最终将该模组设计为可选模组，由用户自行决定是否安装。

或者，你也可以选择安装 [Modern KeyBinding](https://www.curseforge.com/minecraft/mc-mods/modern-keybinding-fabric) 模组，该模组不会与 [Controlling](https://www.mcmod.cn/class/1191.html) 产生冲突。

- 安装了 Amecs 时，通过该模组的接口注册与 Forge 版相同的按键绑定；
- **未安装该模组时，注册不含修饰键的按键绑定。例如，直接按 Y 即可打开模型选择页，无需 Alt**。
