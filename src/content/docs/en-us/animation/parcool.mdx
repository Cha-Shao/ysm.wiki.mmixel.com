The Parcool mod adds various parkour actions for players. Since version 2.2.2, we have added compatibility support for this mod!

<Aside>
  MCMOD Wiki: [Parcool Mod](https://www.mcmod.cn/class/5958.html) Reference
  Video: [Parcool Mod Demo Video](https://www.bilibili.com/video/BV1xJ4m1h7UH)
</Aside>

Some animations require distinguishing between front, back, left, and right directions, so you need to refer to the animation names in the default model animation file `parcool.animation.json`. Also, don't forget to declare this animation file in `ysm.json`!

## Add Animation Mapping File

First, add a type `parcool` in the animation field of `ysm.json`, as shown below:

```json {8}
"animation": {
    "main": "animations/main.animation.json",
    "arm": "animations/arm.animation.json",
    "extra": "animations/extra.animation.json",
    "tac": "animations/tac.animation.json",
    "carryon": "animations/carryon.animation.json",
    // This is the parkour animation
    "parcool": "animations/parcool.animation.json",
    "swem": "animations/swem.animation.json",
    "slashblade": "animations/slashblade.animation.json",
    "tlm": "animations/tlm.animation.json"
}
```

After that, just refer to the animations in the default model's `parcool.animation.json`.

<Aside>
  **If you want to modify the parkour animation, be sure to fill in this
  section. If the modified parkour animation does not work properly, please
  check whether the file is declared here first.**
</Aside>
