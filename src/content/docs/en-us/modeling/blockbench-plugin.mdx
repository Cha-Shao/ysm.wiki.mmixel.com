The YSM model pack format is extremely complex to study, so the simplest and easiest way to get started is to install the ysm-utils Blockbench plugin to create models. It is recommended to use the latest version of Blockbench to install these plugins!

Please select the appropriate plugin to install according to the version of YSM mod you are using:

After downloading, do not change the file name. Choose a suitable location to store the file. Then open Blockbench and select: `File -> Plugins -> Load Plugin from File` to complete the loading of the ysm plugin.

If you need to update the plugin, make sure not to change the plugin name. Simply replace the original plugin file and restart Blockbench.

![6.png](https://s2.loli.net/2023/06/26/HpVFRtY8WCnP25G.png)

When the menu bar shows `是！史蒂夫` or `YSM`, it means the plugin has been successfully loaded.
