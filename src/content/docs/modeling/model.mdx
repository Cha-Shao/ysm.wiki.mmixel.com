## 主模型

主模型的组进行了标准化设计，只要不大改默认模型的组，相关的动画文件基本可以完美兼容。  
在运行游戏后，游戏主目录下的 `config\yes_steve_model\custom\default` 文件夹下会自动生成标准模型和动画文件，你可以在其基础上进行二次修改和分发。

<Aside type="tip">
  为了鼓励和方便大家制作模型包，该 `default` 文件夹的模型以 **CC 0 协议**
  发布，可以自由修改分发。
</Aside>

某些特定组名会被游戏自动赋予特殊功能，**对齐其应用缩放、旋转和位移动画可修改相关渲染的大小、朝向和位置**：

| 组名                                                           | 作用                              | 备注                                                                       |
| :------------------------------------------------------------- | :-------------------------------- | :------------------------------------------------------------------------- |
| `Head`                                                         | 头部定位组                        | 游戏会默认添加头部随视角摆动的动画（历史遗留问题，建议谨慎修改）。         |
| `AllHead`                                                      | 真实第一人称/Better Combat 定位组 | 相关模组中会默认隐藏第一人称此分组下的模型。                               |
| `ViewLocator`                                                  | 视角定位组                        | 真实第一人称模组中，玩家头部高低由此组旋转点的 Y 值决定。                  |
| `LeftHandLocator`                                              | 左手手持物品定位组                | 无此组则不渲染副手物品。                                                   |
| `RightHandLocator`                                             | 右手手持物品定位组                | 无此组则不渲染主手物品。                                                   |
| `ElytraLocator`                                                | 鞘翅定位组                        | 无此组则不渲染鞘翅。                                                       |
| `LeftShoulderLocator` `RightShoulderLocator`                   | 鹦鹉定位组                        | 无此组则不显示肩膀上的鹦鹉。                                               |
| `LeftWaistLocator`                                             | 主手拔刀剑定位组                  | 无此组则不显示主手拔刀剑，**建议与左手手持物品定位组同级放置**。           |
| `RightWaistLocator`                                            | 副手拔刀剑定位组                  | 无此组则不显示副手拔刀剑。                                                 |
| `BladeLocator` <Badge variant="success" text="2.3.0 新增"/>    | 主手拔刀剑刀身定位组              | 用于拔刀剑动画，无此组则不显示刀身，**建议与右手手持物品定位组同级放置**。 |
| `SheathLocator` <Badge variant="success" text="2.3.0 新增"/>   | 主手拔刀剑刀鞘定位组              | 用于拔刀剑动画，无此组则不显示刀鞘，**建议与左手手持物品定位组同级放置**。 |
| `PistolLocator`                                                | 手枪模型定位组                    | 永恒枪械工坊模组中，副手持手枪时渲染腰间手枪模型，无此组则不显示。         |
| `RifleLocator`                                                 | 其他枪械定位组                    | 永恒枪械工坊模组中，副手持步枪/火箭筒时渲染背部模型，无此组则不显示。      |
| `BackpackLocator` <Badge variant="success" text="2.4.0 新增"/> | 背包定位组                        | 用于女仆背包定位（两肩中间），模组背包、拔刀剑/旗帜装饰也参考此组。        |
| 所有以 `ysmGlow` 开头的组（如 `ysmGlowHead`）                  | 发光组                            | 在绝大多数光影下渲染发光效果，**仅作用于当前组**。                         |

## 第一人称手臂模型

<Aside type="tip">
  第一人称手部模型目前暂不支持基岩版动画，但可完美兼容永恒枪械工坊模组。
</Aside>

第一人称视角的手部模型可通过主模型二次修改制作，与主模型共用一套贴图。制作步骤如下：

<Steps>

1. **新建模型**

   新建一个空白的 BlockBench 项目。

2. **复制手臂模型**

   从主模型文件中仅复制 `LeftArm` 和 `RightArm` 分组到该项目的**根目录**下。

3. **调整旋转和位置**

   将 `LeftArm` 和 `RightArm` 分组的 `X`/`Y`/`Z` 旋转角度均设为 `0`（内部分组无需修改，调整后手臂应垂直向下）。

   <Aside type="tip">
     若使用永恒枪械工坊模组时手臂位置异常，可通过向下移动手臂或缩小左右手间距修正。
   </Aside>

</Steps>

此外，在根目录下创建名为 `Background` 的分组，游戏会在第一人称视角将其渲染为背景，不受手持物品或使用状态影响。

## 自定义箭模型

1.2.0 及以上版本支持自定义箭模型，玩家切换模型时，射出的箭（含弩箭）会自动替换为自定义模型，无自定义模型则显示原版箭。

**注意朝向**：在正视图下，箭头应指向左侧。
