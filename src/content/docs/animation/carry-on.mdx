---
title: CarryOn 动画
---

[Carry On 模组](https://curseforge.com/minecraft/mc-mods/carry-on)是一款十分有趣的模组，在按下 Shift 并右键点击的操作下，玩家能够搬起一些方块、生物，甚至是其他玩家。

在 1.16.5 和 1.18.2 版本中，Carry On 模组仅支持搬起玩家和方块，并且会额外渲染一个玩家手臂模型（若要关闭该渲染，需修改 Carry On 模组自身的配置文件）。

**只有 1.19.2 和 1.20 版本的 Carry On 模组才能完美兼容所有功能！**

Carry On 模组的兼容动画存于一个名为 `carryon.animation.json` 的独立文件中，该文件包含四个动画：

| 名称               | 作用                                                                            | 备注     |
| :----------------- | :------------------------------------------------------------------------------ | :------- |
| `carryon:block`    | 玩家抱起方块时的动画                                                            | 循环播放 |
| `carryon:entity`   | 玩家抱起实体时的动画                                                            | 循环播放 |
| `carryon:player`   | 玩家抱起其他玩家时的动画（仅限 1.19.2 和 1.20 版本）                            | 循环播放 |
| `carryon:princess` | 被抱起的那个玩家的动画（仅限 1.19.2 和 1.20 版本）<br/>!!所以动画名叫：“公主”!! | 循环播放 |

## 多余的手臂

若你使用的是 1.18.2 及以前版本的 Carry On 模组，可能会出现如下情况：

![pig.png](https://s2.loli.net/2025/01/30/T7lDLOtSYFPV91o.png)

这是旧版本 Carry On 模组存在的一个 bug，你只需手动修改 `config\carryon-client.toml` 文件，将 `renderArms` 后面的 `true` 改为 `false` 即可：

```toml {6}
#Settings
[settings]
	#If the front of the Tile Entities should face the player or should face outward
	facePlayer = false
	#Arms should render on sides when carrying
	renderArms = true
```
