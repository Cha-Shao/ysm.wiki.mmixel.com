Parcool 模组为玩家增添了各类跑酷动作，自 2.2.2 版本起，我们为该模组添加了兼容支持！

<Aside>
  MCMOD 百科：[Parcool 模组](https://www.mcmod.cn/class/5958.html)
  参考视频：[Parcool 模组演示视频](https://www.bilibili.com/video/BV1xJ4m1h7UH)
</Aside>

由于部分动画需要明确区分前后左右方向，因此你需要参考默认模型动画文件 `parcool.animation.json` 中的动画名称。同时，别忘了在 `ysm.json` 文件中声明该动画文件！

## 添加动画定位文件

首先，你要在 `ysm.json` 的动画字段里新增一个类型 `parcool`，示例如下：

```json {8}
"animation": {
    "main": "animations/main.animation.json",
    "arm": "animations/arm.animation.json",
    "extra": "animations/extra.animation.json",
    "tac": "animations/tac.animation.json",
    "carryon": "animations/carryon.animation.json",
    // 这就是跑酷的动画
    "parcool": "animations/parcool.animation.json",
    "swem": "animations/swem.animation.json",
    "slashblade": "animations/slashblade.animation.json",
    "tlm": "animations/tlm.animation.json"
}
```

之后，参考默认模型的 `parcool.animation.json` 动画即可。

<Aside>
  **若你要修改跑酷动画，务必填写此处。若修改后的跑酷动画无法正常生效，请先检查此处是否声明了该文件**
</Aside>
